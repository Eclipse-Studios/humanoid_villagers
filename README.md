# 👨‍🌾 Humanoid Villagers
*A datapack that replaces those boring villagers with Players! Names, skins and conversations are randomized and can be easily expanded on as it is all data-driven.*

[![fapi-badge](https://cdn.jsdelivr.net/npm/@intergrav/devins-badges@3/assets/cozy/requires/fabric-api_vector.svg)](https://modrinth.com/mod/fabric-api)

*^ only needed if playing on Fabric or Quilt*
